Markdown Syntax

Pelican uses Markdown for content formatting. Here are some basic Markdown syntax examples:

  • Italic text is created using asterisks or underscores.
  • Bold text is created using double asterisks or double underscores.
  • Links can be created using [link text](url).
  • Lists can be created using hyphens, asterisks, or plus signs.
